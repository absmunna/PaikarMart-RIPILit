Build a complete multi-role UI/UX system for a modern multi-vendor platform called "PaikarMart".

SYSTEM TYPE:
- Hybrid platform: Facebook-style product feed + E-commerce marketplace
- Mobile-first + responsive web

ROLES:
1. Guest User
2. Buyer/User
3. Seller/Vendor
4. Admin

-----------------------------------
GLOBAL DESIGN SYSTEM (MANDATORY)
-----------------------------------

Use a futuristic forest green glassmorphism UI:

- Background: deep forest gradient (#061A12 → #0F3D2E)
- Cards: glass effect (blur + transparency)
- Borders: soft white opacity
- Accent: green (#00FF9C) + red (#FF3B3B)
- Avoid full white backgrounds
- Use real product images with white/clean background

-----------------------------------
NAVIGATION STRUCTURE
-----------------------------------

Mobile Bottom Nav:
- Left: Marketplace (shopping UI)
- Center: Home (Facebook-style feed)
- Right: Account

-----------------------------------
ROLE: GUEST USER
-----------------------------------

Access:
- View feed
- View marketplace
- Cannot add to cart / comment

UI Behavior:
- Show "Login Required" on interaction
- Soft CTA prompts

-----------------------------------
ROLE: BUYER (USER)
-----------------------------------

Features:
- Facebook-style product feed
- Like (wishlist), Comment, Share
- Add to cart
- Checkout system
- Order tracking

Feed UI:
- Seller info
- Product image (real)
- Price + discount
- Rating + views
- Action row:
  ❤️ Wishlist
  💬 Comment
  🛒 Add to Cart
  ⭐ Rating
  👁 Views

Marketplace UI:
- Product grid
- Filters
- Categories
- Clean ecommerce layout

-----------------------------------
ROLE: SELLER (VENDOR)
-----------------------------------

Dashboard UI:

Sidebar Navigation:
- Dashboard
- My Products
- Add Product
- Orders
- Earnings

Dashboard Features:
- Stats cards (sales, orders, products)
- Recent orders
- Performance summary

Product System:
- Add/edit/delete product
- Upload real product images
- Set price, stock

Orders:
- Order list
- Status update (Pending → Processing → Delivered)

Earnings:
- Total earnings
- Monthly earnings

UI Style:
- Same glassmorphism theme
- Clean admin-like layout but visually consistent

-----------------------------------
ROLE: ADMIN
-----------------------------------

Admin Panel Features:

- User management
- Seller approval system
- Product moderation
- Order monitoring
- Commission control

Dashboard:
- Total revenue
- Total users
- Active sellers
- Platform analytics

-----------------------------------
MICRO INTERACTIONS
-----------------------------------

- Hover = glow effect
- Click = scale down
- Active = border highlight
- Smooth transitions

STRICT RULES
-----------------------------------

- No flat UI
- No full white screens
- No inconsistent components
- Must reuse component system
- Maintain design consistency across all roles

-----------------------------------
DELIVERABLES

- Modular architecture

Ensure production-ready UI/UX quality similar to top platforms.
